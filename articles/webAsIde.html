The web as IDE

When I asked why people would use Vim or Emacs over a "proper" IDE, the argument of "My OS is my IDE" struck me the most. Personally, I still don't like to use text editors over full-blown IDE's. But the web as IDE seems to be more and more compelling every day. The argument for this case be made best by seeing for yourself. Therefore, I intent to collect the best tools here. This list is entirely subjective, but don't hesitate for a second to tip me off with new stuff not listed here.
