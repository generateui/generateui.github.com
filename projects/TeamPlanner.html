TeamPlanner

A .NET WinForms based tool to plan teams according to availability of employees.

Sensitive information

This article only contains a very small subset of the information of the project. Most information (like the internship report) is packed with privacy sensitive data, and proprietary business processes. This page exists to give you an idea what the project was about, and what the result is. Excusez-moi!

Goal

Develop and design a new "team pipeline" which supports the current scheduling process. This should be done by the Avanade Connected Methods project management method. Information should be fast and easily accessible. Prerequisites for this project are furthermore that it should be developed in Visual Studio Team System, the Microsoft .NET framework, Microsoft Sharepoint and Microsoft Project (Server).
